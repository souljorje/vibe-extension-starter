# Vibe Extension Starter

A tiny beginner-friendly browser extension starter built with Manifest V3.

## Files

- `manifest.json`: extension configuration
- `popup.html`: popup markup
- `popup.css`: popup styles
- `popup.js`: popup behavior
- `background.js`: background service worker
- `blocked.html`: placeholder page for future redirect/block flows
- `blocked.js`: logic for the blocked page
- `icons/`: add extension icons here later

## Load It In Chrome

1. Open `chrome://extensions`
2. Turn on **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `vibe-extension-starter`

The popup already works, so you can click the extension icon and see the starter UI.
